ESTUDOS DE BACK-END e JAVA

Este guia foi criado para estudos de Back-End

Requisitos e Qualificações da Softplan

  1. Conhecimento em JAVA 8 e 11;
  2. Conhecimento em Framework;
  3. Desenvolvimento Front-End com JavaScript;
  4. Conhecimento de princípios de orientação orientados a objetos;
  5. Experiencia com Banco de Dados
    1. Oracle;
    2. SQL Server;
    3. Postgre SQL;
  6. Práticas Clean Code;
  7. Princípios SOLID;

Responsabilidades e Atribuições

  1. Product Owner (PO);
    1. Realizar Estimativas;
    2. Planejar e Participar das Sprints;
      1. Em projetos de Produção;
      2. Em projetos de sustentação;
  2. Desenvolver Funcionalidades;

    3. Funcionalidades descritas em épicas e histtórias de usuários.

  3. Criar Estruturas;

    4. Realizar a criação de estruturas organizadas e eficientes, para suportar novas funcionalidades, atendendo as necessidades dos clientes e realizando integrações.

  4. Identificar e Corrigir Bugs;

    5. Realizar as correções de bugs, garantindo a estabilidade e performance do software.

    1. Solicitadas pelos clientes;
    2. Solicitadas pela equipe de qualidade;
  5. Inserir e Gerenciar Códigos;

    6. Gerenciamento do repositório de armazenamento do código fonte, deverá ser mantido a integridade e organização do projeto.
